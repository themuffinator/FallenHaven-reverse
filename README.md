# FallenHaven-reverse
An experiment at reverse-engineering the Fallen Haven game binaries using agentic AI.

## Documentation index
- `documents/game-file-structure.md`: inventory of `package/` directory layout and root-level installer assets.
- `documents/file-types.md`: observed structures for text-like formats plus notes on binary formats.
- `documents/executable-analysis.md`: static observations from the main executable (`FHDEMO.EXE`).
- `documents/supporting-binaries.md`: installer/support binary notes (InstallShield/NE format).

## Reference material scope
The game data under `package/` is **reference-only**. Do not modify any files in that directory.
