# AGENTS

## Repository purpose
This repository documents and reverse-engineers the Fallen Haven demo binaries and data to support understanding, QoL, and compatibility improvements.

## Game data location
The game files are located under `package/` and are **reference-only**. Do not modify files inside `package/`.

## Reverse engineering workflow
- Always strive to update your understanding of the file as you work through a skills file, updating/creating mapping files and anything else which assists in providing clarity over the code.
